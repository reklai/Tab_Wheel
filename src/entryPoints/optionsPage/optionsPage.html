__EXTENSION_NAME__ — Settings

TabWheel

Modifier-scroll cycling for tagged browser tabs.

Alt + WheelCycle tagged tabs, or all tabs when none are tagged.
Alt + TOpen the in-page tagged-tab panel.
Alt + MOpen the TabWheel help menu.
Alt + Left ClickTag the current tab.
Alt + Right ClickRemove the current tab tag.
Alt + Middle ClickConfirm before clearing all current-window tags.
