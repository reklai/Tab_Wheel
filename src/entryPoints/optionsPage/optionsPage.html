__EXTENSION_NAME__ - Settings

TabWheel

Reliable modifier-wheel cycling with MRU mode, click gestures, and toolbar fallbacks.

Alt + WheelCycle tabs using the current cycle mode.
Alt + Left ClickOpen the in-page search launcher.
Alt + Middle ClickJump to the most recently used tab.
Alt + Right ClickClose the current tab and return to the most recently used tab when available.
Toolbar popupUse fallback buttons, change mode, and tune wheel behavior.
Modifier-click caveat Some sites and browsers reserve modifier + left/middle/right click combinations for opening links, saving links, selection, context menus, or system shortcuts. If one combo conflicts on your setup, choose another modifier or require Shift.
Extension constraints Page gestures run on normal web pages. Browser UI and internal pages such as chrome://extensions, about:addons, the web store, PDF viewers, and some restricted pages may block content scripts; use the toolbar popup controls there.
