__EXTENSION_NAME__ - Settings

TabWheel

Reliable modifier-wheel cycling with a toolbar fallback and optional Wheel List mode.

Alt + WheelCycle tabs using the current cycle mode.
Alt + Left HoldSwipe left to tag/untag or right to change cycle mode.
Alt + Right HoldSwipe up for last recent, down-left to close current, or down-right for a new tab.
Alt + Middle ClickOpen the in-page command panel.
Toolbar popupUse fallback buttons, change mode, and manage Wheel List entries.
Modifier-button caveat Some sites and browsers reserve modifier + mouse-button combinations for opening links, saving links, selection, context menus, autoscroll, or system shortcuts. If one combo conflicts on your setup, choose another modifier or require Shift.
Extension constraints Page gestures run on normal web pages. Browser UI and internal pages such as chrome://extensions, about:addons, the web store, PDF viewers, and some restricted pages may block content scripts; use the toolbar popup controls there.
